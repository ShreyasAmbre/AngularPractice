Date pipe is used to trasnform date in proper format as per requirements

{{myDate}} this is orginal value of date property

{{myDate | date}} here only date pipe is use with no parameter

{{myDate | date:'short'}} here only date pipe is use with 'short' parameter

{{myDate | date:'medium'}} here only date pipe is use with 'medium' parameter

{{myDate | date:'fulldate'}} here only date pipe is use with 'fulldate' parameter
